# About MeshCore Canada

MeshCore Canada is an independent community project that helps people across
Canada find local MeshCore groups, choose and configure equipment, operate
shared network services, and improve the public documentation together.

!!! important "Independent community project"
    MeshCore Canada is not affiliated with, endorsed by, or officially
    connected to the MeshCore or MeshOS projects. For upstream firmware and
    protocol information, use the [official MeshCore documentation](https://docs.meshcore.io/){ target="_blank" rel="noopener" }.

## What the project maintains

- task-based setup, hardware, and troubleshooting guides;
- the Canadian community directory and region definitions;
- shared observer and MQTT integration guidance;
- links to the forum, Discord, and public network tools; and
- the open source for this website in the
  [MeshCore Canada repository](https://github.com/MeshCore-ca/MeshCore-Canada){ target="_blank" rel="noopener" }.

## Stewardship and accountability

Responsibility is split by the thing being changed. This avoids presenting one
person as the authority for every part of the project.

| Stewardship area | Responsibility |
|---|---|
| Site and documentation maintainers | Review site changes, evidence, accessibility, and releases. |
| Service operators | Operate the community's shared web, MQTT, and live-data services and publish service-specific guidance. |
| Region maintainers | Protect the national region data model and release checks. |
| Provincial, territorial, and local stewards | Review community listings, names, settings, and local boundary proposals. |
| Contributors | Report problems, improve guides, and provide reviewable evidence. |

The repository history and public review record are the durable source for
website decisions. A directory listing, service credit, or past contribution
does not by itself grant authority over another area.
